hint=help

Usage

    foo.source=/path/to/helproot
    foo.files=foo.html
    foo.hint=help
    INSTALLS+=foo

Description

Register a help install. This uses installhelp so that both untranslated help files (<helproot>/html/*) and translated help files (<helproot>/<lang>/html/*) are installed.

See also Install Hints.
