hint=sxe

Usage

    foo.files=mybin
    foo.path=/bin
    foo.hint=sxe
    foo.domain=window
    INSTALLS+=foo
    qtopia_project(external app)
    target.hint=sxe
    target.domain=window

Description

Register an app/script with sxe_installer. For most projects you won't need this because target.hint gets this by default for Qtopia projects. This is most useful when installing additional scripts or non-Qtopia projects.

Note that for compatibility with older code pkg.domain will be used if target.domain is empty.

See also Install Hints, qtopia_project(), no_target_sxe, and Packages (overview).
