no_general_pri

Usage

    CONFIG+=no_general_pri

Description

Set this value to prevent loading general.pri. This is useful for device profiles that want to heavily customize the project list. They can simply list the projects they want instead of having to list the projects they do not want.

See also CONFIG Values and Device Profiles.
