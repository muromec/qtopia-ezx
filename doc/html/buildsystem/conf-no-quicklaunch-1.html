no_quicklaunch

Usage

    CONFIG+=no_quicklaunch

Description

Notify the build system that your application should not be built for quicklaunching. This is only applicable if you have used the qtopia_main CONFIG value.

Note that the configure switch -force-quicklaunch overrides this value.

Note that packages cannot contain quicklaunched applications. See Package Limitations for more informtation.

See also CONFIG Values, Applications, qtopia_main, and Quicklauncher.
