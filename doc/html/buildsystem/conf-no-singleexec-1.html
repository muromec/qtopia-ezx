no_singleexec

Usage

    CONFIG+=no_singleexec
    idep(CONFIG+=no_singleexec,CONFIG)

Description

Notify the build system that your application should not be built into the singleexec binary. This is only applicable if you have used the qtopia_main CONFIG value. Libraries should use idep() so that their depenants are also excluded from the build.

See also CONFIG Values, Applications, qtopia_main, idep(), and Qtopia Single Exec.
