Messaging

Introduction

Qtopia provides both a client application supporting common messaging functionality, and a library for developing other applications that work with messages.

The Messages client application provides SMS, MMS and email messaging services, while the Qtopia Messaging Library provides access to all messages stored on a device, via a uniform interface.

For detailed information refer to:

  • Qtopia Messaging Library
