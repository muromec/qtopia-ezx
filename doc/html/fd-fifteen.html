Content: Games: Fifteen
Functional Description

The objective of Fifteen is to place the pieces in order from left to right, top to bottom by sliding them around.

Features

  • Start a Game
  • Use an Image as Pieces

Start a Game (QTOPIA-520)

The user is presented with a set of pieces. If the user has completed the game before, the pieces are shuffled. If not, the user starts the game from where the user stopped.

Use an Image as Pieces (QTOPIA-521)

The user is able to use an image stored on the device instead of the standard set of plain numbered pieces.
