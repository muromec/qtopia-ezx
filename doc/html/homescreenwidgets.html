Detailed Description

The HomeScreenWidgets class is a factory for homescreen widgets.

The HomescreenWidgets class is used as a factory for creating homescreen widgets that have registered. Homescreen widgets are registered with the HSWIDGET macro, which takes the name of the widget and the method used to create the widget. Typically this will be the name of the widget class and the constructor of the class, for example:

    HSWIDGET(WorldmapHSWidget, WorldmapHSWidget)

This class is part of the Qtopia server and cannot be used by other Qtopia applications.

See also AnalogClockHSWidget, AppointmentsHSWidget, LauncherHSWidget, and WorldmapHSWidget.
