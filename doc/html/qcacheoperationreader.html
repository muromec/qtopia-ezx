Detailed Description

The QCacheOperationReader class reads WAP Cache Operation (CO) documents

The QCacheOperationReader class extends QWbXmlReader to provide support for reading documents in the WAP Cache Operation (CO) language. This language is defined in the WAP standard wap-175-cacheop-20010731-a.pdf.

See also QWbXmlReader.
