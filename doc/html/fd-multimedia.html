Multimedia
Functional Description

Qtopia includes a multimedia framework for applications that playback music and videos. Features include simple and abstracted multimedia playback APIs together with reusable control widgets. The reference integration of the Helix DNA Client codec library is provided with Qtopia and supports development for various file system, file format and datatype plug-ins. Simplified APIs provide simple playback interfaces and avoid the requirement to call into the Helix DNA Client directly.

Features

  • Common Application Framework
  • GStreamer Integration
  • MediaServer
  • Qtopia Multimedia Architecture

Common Application Framework (QTOPIA-870)

The multimedia subsystem provides a common framework for applications that playback music and videos. The common application framework is the basis of the Media Player application. Some features of the framework include:

  • Multimedia playback API
  • Control widgets

GStreamer Integration (QTOPIA-876)

The media system has the ability to play audio and video with the GStreamer framework. GStreamer itself is not supplied with Qtopia, but rather must be built as part of the device platform onto which Qtopia is integrated.

MediaServer (QTOPIA-873)

Qtopia's MediaServer provides an integration point for Media Frameworks.

Media Frameworks handle the decoding of audio and video.

The MediaServer supports the following client APIs

  • QSound
  • QtopiaMedia

Qtopia has a Media Framework that uses plugins to play audio files.

Qtopia Multimedia Architecture (QTOPIA-872)

  • Modular Low Level Architecture
    • Highly modular architecture of reusable components
    • Extensibility through pluggable components
      • Codecs, acceleration, drivers etc.
  • High Level APIs for easy access to common multimedia functions e.g. content playback and capture
  • Integrate with a standards compliant DRM implementation
