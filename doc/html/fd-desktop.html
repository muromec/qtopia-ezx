Outlook Synchronization (QTOPIA-416)

Qtopia Sync Agent

  • Can sync data from calendar, contacts and tasks with Microsoft Outlook
  • Tested configurations:
    • Outlook 2002 with the latest service pack, with Windows 2000
    • Outlook 2003 with the latest service pack, with Windows XP
    • Outlook 2007 with the latest service pack, with Windows Vista
  • Tested only on 32-bit Windows 2000/XP/Vista, not 64-bit Windows
  • Tested only with administrator user accounts
  • Installs in system tray, no significant UI
  • Requires some form of serial or TCP/IP driver for connection to device
    • these could be provided by Bluetooth or Wifi, but there is no explicit support for these technologies
    • support for direct access to the device data (device acts as USB mass storage) is not present
  • Installer automatically requests administrator privileges.
  • Communication with Qtopia uses a simple line-oriented protocol.
  • Tested with Qtopia running in QVFb (using ethernet and a real serial cable) and the Greenphone (using USB-serial).
  • The sync communications and algorithm are proprietary (not SyncML etc.).
  • Design described in Desktop Synchronization Overview

Greenphone Sample Implementations

  • Greenphone serial driver also installs on 32-bit Windows 2000/XP/Vista.
  • The Greenphone shows up as serial port COM20 but due to driver limitations special code must be enabled (the GUI has a check box for this).
  • The Greenphone must be set to serial mode (using Applications -> Synchronization) to talk to Qtopia Sync Agent using the serial driver.
