Welcome to Qtopia

Overview

This is Trolltech's Qtopia Open Source edition. See the LICENSE file for the licensing information of this software.

This package contains source code for Qtopia plus parts of the source code of Qt. Apart from this source package it should not be necessery to install any other Trolltech package to be able to compile Qtopia.

Documentation

The package contains the Qtopia reference documentation including all relevant Qt documentation. The documentation includes the following:

  • Qtopia documentation index: the starting page for all documentation.
  • Getting Started page: for help with installing, building and running Qtopia.
  • Guide to Configuring and Building Qtopia: for detailed help with building Qtopia from source.

For the most up-to-date information, please refer to the Trolltech Documentation Site.
